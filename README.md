## 量子色域 — 视界
专注于 Minecraft UI 美化与渲染优化的核心模组，旨在通过量子级视觉处理技术，提供沉浸式、高自定义的界面体验。

### 功能:

1. 动态模糊渲染
2. 自定义 Tooltips
3. 多风格 Top 栏
