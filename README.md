## 量子色域 — 视界
专注于 Minecraft UI 美化与渲染优化的核心模组，旨在通过量子级视觉处理技术，提供沉浸式、高自定义的界面体验。支持主流 Forge/Fabric 加载器，开源协议为 MIT，可供开发者二次开发与公众使用。

### 功能:

1. 动态模糊渲染
2. 自定义 Tooltips
3. 多风格 Top 栏
